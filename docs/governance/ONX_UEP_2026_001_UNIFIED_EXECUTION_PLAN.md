## 4) Weekly Gates

Weekly gate template:

1. Demo (10 min)
2. Build check: `bun run build` (2 min)
3. Test check: `bun run test` (2 min)
4. Lint check: `bun run lint` (2 min)
5. Evidence review (`EV-*`) (5 min)
6. Founder decision: PASS / FAIL / CONDITIONAL PASS

Decision protocol:

- PASS: proceed to next week
- FAIL: 2-day remediation buffer then re-gate
- CONDITIONAL PASS: proceed with explicit issue ledger
